Introduction

    Internet est un réseau mondial d'ordinateurs qui permettent aux gens de communiquer.
    Le web est une application qui fonctionne sur internet et permet aux utilisateutrs d'accédera des documents hypertexte via un navigateur.
    Les URL sont des adresses uniquent qui identifient l'emplacement des recoures sur le web. Dans word, vous pouvez créer des liens hypertexte en sélectionnant le texte que vous souhaitez utiliser comme lien, en cliquant sur "insertion",puis entrant l'URL de la page que vous souhaiter lier.

Aspects techniques du Web

Le Web utilise trois mécanismes clés: les URL pour localiser les ressources, le protocole HTTP pour les échanges client-serveur, et le langage HTML pour décrire les pages Web avec du texte, des images et des liens hypertextes. HTTP est sans état, ce qui signifie qu'il n'a pas de mémoire des transactions précédentes. Le HTML utilise une approche de programmation déclarative pour décrire les pages Web.

Page web et site web:

Une page Web est un document multimédia avec des textes, des images et des interactions, stocké dans un fichier HTML, qui peut être interprété et affiché par un navigateur en utilisant une URL. Les liens hypertextes permettent de naviguer entre les pages Web d'un site Web, qui est un ensemble de pages Web organisées autour d'une thématique commune, reliées entre elles et accessibles depuis une page d'accueil commune et une URL de base commune. Les sites Web sont généralement hébergés sur un même serveur et structurés sous forme d'arborescence de dossiers et de fichiers, utilisant des chemins relatifs.

Création d'une page web HTML:

HTML est un langage qui permet de créer des pages Web. Ces pages sont constituées d'éléments de code appelés balises, qui sont interprétées par un navigateur Web pour afficher le contenu d'une page. Les balises minimales<html>, <head >, <title > ,<body > , sont nécessaires pour créer un document HTML, et on peut également utiliser des balises pour définir les titres et les paragraphes. Pour créer un document HTML, on peut utiliser un éditeur de pages Web, tel que Dreamweaver, ou un éditeur de texte, tel que le Bloc-notes ou WordPad.
